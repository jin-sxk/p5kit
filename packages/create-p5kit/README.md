# create-p5kit

Create a p5kit app from a starter template.

```sh
npm create p5kit my-sketch
cd my-sketch
npm install
npm run dev
npm run doctor:android
npm run build:ios
```

p5kit projects are p5.js/Vite sketches with Capacitor already configured for
iOS and Android handoff.

Generated projects include `npm run doctor:android` for checking Java and
Android SDK setup before Android build/run commands.

p5kit is not affiliated with the Processing Foundation or p5.js.
